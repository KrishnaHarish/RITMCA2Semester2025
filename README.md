## üõ†Ô∏è Quick Start Guide

### Prerequisites
- **Node.js** (v16+) for web applications
- **MongoDB** (v4.4+) for database
- **Java JDK** (v11+) for Java programs
- **Python** (v3.8+) for network programming

### Running the Mini Projects

1. **Choose a project** from the mini projects table above
2. **Navigate to the project directory**:
   ```bash
   cd cooking-blog        # or crop-yield-prediction or recipe-management
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Set up environment**:
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```
5. **Start the application**:
   ```bash
   npm run dev            # Development mode
   npm start              # Production mode
   ```

### Running Java Programs
```bash
javac ClassFundamentalsDemo.java
java ClassFundamentalsDemo
```

### Running Python Network Programs
```bash
# Start server first
python TCP-Server.py

# In another terminal, run client
python TCP-Client.py
```
